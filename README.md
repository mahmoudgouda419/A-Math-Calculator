# Multi-Use Calculator

A responsive glassmorphism multi-use calculator built with HTML, CSS, and JavaScript. It features a standard calculator, tip calculator, and live currency converter with real-time exchange rates connected with frankfurter api.

## Features

- Standard Calculator
- Tip Calculator
- Live Currency Converter
- Responsive Design
- Glassmorphism UI

## Built With

- HTML
- CSS
- JavaScript
- Frankfurter API

## Run Locally

Clone the repository and open `index.html` in your browser, or use Live Server for the best experience.

```bash
git clone https://github.com/mahmoudgouda419/A-Math-Calculator.git
```
